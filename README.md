# Honest Graphics Website

Static marketing website for Honest Graphics.

## Run locally

Serve the project root with any static server, then open:

`/html/honest-graphics.html`

Example:

```bash
python3 -m http.server 5501
```

Open:

`http://127.0.0.1:5501/html/honest-graphics.html`

## Quick Quote behavior

- validates required fields
- user can choose `Send via WhatsApp` or `Send via Email`
- both options prefill enquiry details from the form
